# Vulnerability Scanning Techniques

### Scan Types
In VAPT there exist different scans for different types of things we perform into, network, application, infrastructure. Automated tools nmap, zap, nikto, etc..

Later it is required to manually check over the of these scans check if it's true or not. So from the result of the scan we check over false positives and true positives.
Guidelines and Standard like NIST SP 800-115, OWASP Testing Guide helps in making it a structured way of performing scans.
